# C++

### 基础

#### 关键字

##### const

define 是 预处理阶段，const是编译和运行阶段

define是替换字符串，const 是修饰了变量的

##### static

1. 普通变量，main函数运行前便分配空间，在静态区
2. 普通函数，只定义的文件内可用
3. 成员变量，类访问
4. 成员函数，只能访问static的成员变量

##### this

隐含每一个非静态成员函数中，指向调用对象

成员函数调用前，对象地址编译时被赋给this，每次调用成员函数都隐式使用this

当一个成员函数被调用时，自动向它传递一个隐含的参数，该参数是一个指向这个成员函数所在的对象的指针。

this 指针的类型为：ClassName* const，`const` 成员函数中，this 指针的类型为：const ClassName* const

是个右值，所以不能取得 `this` 的地址（不能 `&this`）
